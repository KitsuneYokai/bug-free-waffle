  FAQ

What is this site?

This site is a list for Osu! private servers (3rd party servers).
You can vote for your favorite server and write a review for it.

What do I do now?

Just click on one server card from above and you will be redirected to a detailed server view.
On the server view you can see some information about the server (Discordserver / website / reviews /...)

How do i connect to a server?

You have to add -devserver <serverurl>  to your Osu! shortcut. I will add a download button for the server shortcut later. So you don't have to do this manually ;)
